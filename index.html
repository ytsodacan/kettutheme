Export & Install

Themes are installed in Kettu → Settings → Themes by pasting a raw JSON URL. Click the button below to instantly host your code and get a link.

Manual Instructions
1
Host the downloaded .json file on any public URL (GitHub raw, jsonbin, paste.gg, etc.)
2
Open Discord with Kettu / Bunny installed, go to Settings, and scroll down to Themes.
3
Tap the + button and paste your raw JSON URL. Apply the theme!
